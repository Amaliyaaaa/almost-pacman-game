# almost-pacman-game
Это небольшая игра, напоминающая pacman - первое python-приложение, которое я создала в самом начале своего пути в программировании, когда изучала основы Python.
В этом проекте я не использовала сложные инструменты, движок - pygame.
Запустить можно, скачав exe-файл. Цель игры - добраться до цели (сердечки в правом нижнем углу) и не погибнуть.
### Управление:
Передвижение - стрелки вверх, вниз, влево, вправо.
Выстрел - пробел.
